🚀 GETTING STARTED (3 SIMPLE STEPS)
═════════════════════════════════════════════════════════════════════════════════

Step 1: Install Dependencies & Compile
───────────────────────────────────────
Run one of these commands:

  Option A (Automated):
    bash get-started.sh

  Option B (Manual):
    npm install
    npm run compile

Step 2: Launch Debug Environment
─────────────────────────────────
  • Press F5 in VS Code
  • Or use: Debug → Start Debugging
  • A new VS Code window opens with your extension active

Step 3: Test the Extension
──────────────────────────
  1. Open examples/example.one (or .hcl or .arc)
  2. See syntax highlighting in action
  3. Type snippet prefixes:
     - one:if, one:for, one:function (ONE)
     - hcl:resource, hcl:variable (HCL)
     - arc:def, arc:loop, arc:class (ARC)
  4. Press Shift+Alt+F to format code
