# ONE Icon Theme

A beautiful VS Code icon theme with custom icons for ONE template files (.one), featuring Blade-inspired design and the signature ONE red (#FF2D55) color.

## Features

- 🎨 Custom icon for `.one` files with Blade-inspired design
- 🔴 Signature ONE color (#FF2D55) for easy visual identification
- 📦 Lightweight and performant
- 🎯 Seamless integration with VS Code

## Installation

1. **Via VS Code Extension Marketplace**
   - Search for "ONE Icon Theme" in VS Code Extensions
   - Click Install

2. **Via VSIX File**
   - Download `one-icon-theme-1.0.0.vsix`
   - Run: `code --install-extension one-icon-theme-1.0.0.vsix`

## Usage

After installation:

1. Open VS Code Settings
2. Go to **File** → **Preferences** → **File Icon Theme** (or use **Cmd+K Cmd+T** on macOS)
3. Select **ONE Icons** from the dropdown

## Icon Design

The ONE icon features a minimalist Blade-inspired design:
- File document shape with #FF2D55 fill
- Blade pattern at top and bottom
- Center diamond accent for depth

## Compatibility

- VS Code 1.75.0 or higher
- All operating systems (macOS, Windows, Linux)
- Compatible with all VS Code color themes
