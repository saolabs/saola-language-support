# Changelog

All notable changes to the Template Languages extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-30

### Added
- Initial release of Template Languages extension
- Support for `.one` file extension with syntax highlighting
- Support for `.hcl` file extension with syntax highlighting
- Support for `.arc` file extension with syntax highlighting
- Comprehensive snippets for each language:
  - ONE: if/else, loops, functions, classes, variables
  - HCL: resources, variables, modules, outputs, providers
  - ARC: if/then/else, loops, definitions, classes, traits
- Document formatting provider for all three languages
- Language configuration for proper bracket matching and indentation
- Emmet support compatibility
- Comment highlighting for all languages

### Features
- Syntax highlighting with TextMate grammar definitions
- Smart code indentation based on block structure
- Snippet templates with placeholders for quick coding
- Auto-closing bracket configuration
- Language-specific formatting rules

### Repository
- GitHub repository setup
- MIT License
- Comprehensive README with usage examples
